Lesson 9.

DOM - Document Object Model

Все элементы на странице можно представить в виде глобального обьекта document

Чтобы находить элементы в DOM нужно добавить чтото им чтото уникальное, какуюто отличительную особенность, например класс или id:

  • По тегам- getElementsByTagName
  • По классам- getElementsByClassName
  • По id- getElementById
  • document.querySelector(".method__list")-универсальный метод поиска. Класс уточняется серез .,id через #. При этом в ответе прийдет первый элмент удовлет поиску

1 способ подключается вконце перед закрывающимся тегом
