Grid layot

Grid это один из основных инструментов позиционирования в CSS Дочерние Элементы в Grid контейнере позиционируюсяв сетке, которой мы задаем правила для колонок и строк(grid-сетка) ()

  • grid-template-columns- шаблон- схема построения колонок в grid
  • gap- промежуток жмежду элементами
  • grid-template-rows размер добавленных строк
  • repeat auto-fill, minmax(100px, 1fr) ячейки будут рястягиваться пока не получится добавить новую с минимальным размером 100px
  • grid-template-areas - grid схема , в которой можно обьединять нескольк ячеек в одну и группировать их, назначать в них элементы через обращеие к селекторам через grid-template-areas, указав зону area в строчном значении
