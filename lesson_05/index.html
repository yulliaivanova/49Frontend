Lesson 5

Свойство position определяет , какой будет использован метод для расположения элемента (static, relative, fixed, absolute или sticky)

  • static- Свойство position по умолчанию для всех элементов, когда элементы ведут себя привычно для нас без изменений
  • fixed- вырывает элементы с потока и делает его невидимым для остальных элементов
  • sticky- приклеивает элемент, сохраняя за собой местои вырывается из потока при указании значения координат
  • relative-вырывается из потока, но оствляет за собой место и сдвигается относительно него. Обычно используется в связке с absolute, чтобы указывать позиционирование для родителя -не влияя на свойства элемента
  • absolute - по умолчанию свойство двигает элементы относительно body через свойства left, top,right, bottom. Если мы хотим двигаться относительно другого элемента и сгрупироваться с ним -мы позиционируем его родителя , с которым произойдет группировка
